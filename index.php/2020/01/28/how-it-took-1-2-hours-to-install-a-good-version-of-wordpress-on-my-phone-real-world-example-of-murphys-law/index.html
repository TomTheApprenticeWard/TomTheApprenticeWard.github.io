How It Took 1-2 Hours To Install A Good Version Of WordPress On My Phone (Real World Example Of Murphys Law)

So basically I use WordPress on my phone as a primary component of my eBay listing workflow. However the latest version of wordpress has some issues which make it not work very well in my workflow. so because I had backed up an earlier good version of wordpress using titanium backup I decided to try to restore it. However that didn’t work at first and a number of other things I tried it didn’t work. So this is basically a real world example of Murphy’s law. Just about everything that could go wrong did go wrong.
